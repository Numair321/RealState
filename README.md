# Real Estate MLM Platform

A comprehensive Multi-Level Marketing (MLM) real estate platform with role-based access control.

## Project Structure

```
├── frontend/          # React + Tailwind CSS frontend application
└── backend/           # Node.js backend API (to be created)
```

## Features

### Multi-Level Marketing (MLM) Structure
- Referral system with multi-level commission sharing
- Network growth tracking
- Team building and management
- Dual earning model (direct transactions + network activities)

### Property Listing Platform
- Multiple property categories (Residential, Commercial, Industrial, Land)
- Advanced search and filtering
- City-wise and location-wise organization
- Hot deals section
- Direct owner listings

### Intelligent Lead Management
- Automated geographic-based lead distribution
- Smart escalation system
- Complete interaction history
- Follow-up scheduling
- Performance analytics

### User Roles
- **Admin**: Platform oversight, associate management, commission configuration
- **Associate**: Lead management, team building, earnings tracking
- **Company**: Team management, regional analytics
- **Customer**: Property search, saved properties, requirements posting

### Commission Management
- Configurable multi-level commission rates
- Transaction-based commissions
- Referral bonuses
- Real-time tracking and transparency
- Automated calculation and distribution

## Getting Started

### Frontend Setup

```bash
cd frontend
npm install
npm run dev
```

The frontend will run on `http://localhost:3000`

See `frontend/README.md` for detailed frontend documentation.
